Einführung

JavaScript Garden ist eine wachsende Sammlung von Erklärungen der verzwicktesten Teile von JavaScript. Es gibt Hinweise um häufige Fehler, Performance Probleme und schlechten Stil zu vermeiden.

JavaScript Garden ist keine Anleitung um JavaScript zu lernen. Ein grundlegendes Verständnis der Sprache wird wärmstens empfohlen. Eine gute Einführung findet sich zum Beispiel im Mozilla Developer Network.
